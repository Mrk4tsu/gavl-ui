# Layout

Utility layout của GAVL UI được thiết kế để bạn dựng khung nhanh mà vẫn giữ nhịp pixel UI nhất quán. Thay vì viết CSS mới cho từng block nhỏ, bạn có thể ghép các class layout có sẵn để tạo container, row, grid, spacing và alignment chỉ trong vài dòng HTML.

## Gợi ý sử dụng

- Dùng utility layout để dựng khung nhanh, sau đó mới thêm component class như `card`, `form`, `badge`.
- Chọn `gap` cho khoảng cách giữa item trong cùng một group; chọn `margin/padding` cho khoảng cách giữa các block hoặc khung nội dung.
- Với nội dung dài, ưu tiên `container-sm` hoặc `container-xs` để giảm độ dài dòng và dễ đọc hơn.
