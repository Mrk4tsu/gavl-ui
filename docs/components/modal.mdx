## Anatomy

| Phần tử | Class | Vai trò |
|---|---|---|
| Backdrop | `.gv-modal-backdrop` | Lớp phủ toàn màn hình, khóa focus thị giác vào modal |
| Container | `.gv-modal` | Hộp modal chính |
| Header | `.gv-modal-header` | Chứa title và close action |
| Title | `.gv-modal-title` hoặc `.gv-modal-title-pixel` | Tiêu đề modal |
| Body | `.gv-modal-body` | Nội dung chính |
| Footer | `.gv-modal-footer` | Hành động cuối modal |

## Ghi chú triển khai

- Modal chỉ là CSS shell; trạng thái mở/đóng cần JavaScript của ứng dụng quản lý.
- Khi dùng thực tế, nên khóa scroll nền và trap focus khi modal mở.
- `.gv-modal-close` là nút giao diện, nhưng hành vi đóng modal vẫn cần JS của bạn xử lý.
