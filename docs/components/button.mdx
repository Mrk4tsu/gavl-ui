# Button

`Button` là component hành động cơ bản nhất trong GAVL UI. Nền tảng của button là `.gv-btn`, sau đó ghép thêm variant, size hoặc behavior class theo nhu cầu.

## Cấu trúc cơ bản

```html
<button class="gv-btn gv-btn-primary">Bắt đầu</button>
```

    <button type="button" className="gv-btn gv-btn-link">Link</button>
  </div>

| Class | Mục đích |
|---|---|
| `.gv-btn-primary` | CTA chính, hành động nổi bật |
| `.gv-btn-secondary` | Hành động phụ |
| `.gv-btn-dark` | Nút trên nền sáng cần tương phản mạnh |
| `.gv-btn-danger` | Xóa, hủy, hành động phá hủy |
| `.gv-btn-success` | Xác nhận, hoàn tất |
| `.gv-btn-info` | Thông tin, điều hướng mềm |
| `.gv-btn-purple` | Accent phụ, special action |
| `.gv-btn-ghost` | Hành động nhẹ |
| `.gv-btn-link` | Hành vi giống text link |
