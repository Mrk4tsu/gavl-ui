# Typography

Trang này tập trung vào cách dùng typography utility trong thực tế: chọn scale nào cho đúng ngữ cảnh, khi nào dùng pixel font, khi nào nên chuyển sang body font để tăng độ đọc, và cách ghép text color utility để nội dung rõ ràng hơn.

Nếu bạn muốn xem token đầy đủ của font, size và line-height, xem thêm [Design Tokens](/docs/getting-started/design-tokens). Trang này ưu tiên góc nhìn sử dụng thực tế.

## Gợi ý sử dụng

- Dùng `display` cho text ngắn và thiên về trang trí; với câu dài, body font vẫn dễ đọc hơn.
- Với heading tiếng Việt, ưu tiên `.gv-heading-*` hoặc ghép thêm `.gv-vi-tight`.
- `gv-text-md` là default tốt nhất cho docs paragraph; chỉ tăng lên `lg/xl` khi cần nhấn mạnh.
- `gv-caption`, `gv-text-xs`, `gv-text-muted` hợp cho metadata, nhưng không nên dùng làm nội dung chính.
