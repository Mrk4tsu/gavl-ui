# Form

GAVL UI cung cấp bộ class cho input text, select, textarea, helper text, error state và input group.

## Input cơ bản

<div className="docs-demo">
  <div className="docs-form-grid docs-form-grid--two">
    <div className="gv-form-group">
      <label className="gv-form-label" htmlFor="demo-email">Email</label>
      <input id="demo-email" className="gv-input" type="email" placeholder="you@example.com" />
      <div className="gv-form-helper">Helper text mặc định.</div>
    </div>
    <div className="gv-form-group">
      <label className="gv-form-label" htmlFor="demo-name">Tên hiển thị</label>
      <input id="demo-name" className="gv-input gv-input-success" type="text" defaultValue="GAVL Player" />
      <div className="gv-form-helper">Input với state thành công.</div>
    </div>
  </div>
</div>

```html
<input class="gv-input" type="text" placeholder="Nhập nội dung" />
<input class="gv-input gv-input-success" type="text" value="Hợp lệ" />
```

## Ghi chú

- `.gv-form-label-vi` phù hợp hơn khi label chứa tiếng Việt dài.
- Modal, submit, validation state vẫn cần JavaScript của ứng dụng để điều khiển hành vi.
