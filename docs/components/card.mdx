# Card

`Card` là panel nền tảng để gom nội dung thành từng khối rõ ràng. Trong GAVL UI, card không chỉ là một hộp chứa text, mà còn là pattern dùng cho quest item, stat panel, dashboard block, reward card hoặc content section có hành động đi kèm.

Base class là `.gv-card`. Từ đó bạn ghép thêm:
- `.gv-card-dark`, `.gv-card-gold` cho biến thể màu
- `.gv-card-hover` cho hiệu ứng nhấc card khi hover
- `.gv-card-header`, `.gv-card-body`, `.gv-card-footer`, `.gv-card-image` cho cấu trúc nội dung

## Anatomy

| Phần tử | Class | Vai trò |
|---|---|---|
| Container | `.gv-card` | Khối card chính |
| Variant | `.gv-card-dark`, `.gv-card-gold` | Đổi theme màu của toàn card |
| Hover | `.gv-card-hover` | Thêm hiệu ứng nổi lên khi rê chuột |
| Header | `.gv-card-header` | Title, badge, action phụ |
| Body | `.gv-card-body` | Nội dung chính |
| Footer | `.gv-card-footer` | CTA hoặc metadata cuối |
| Media | `.gv-card-image` | Ảnh đầu card |
| Pixel title | `.gv-card-title` | Tiêu đề kiểu pixel |
| Vietnamese title | `.gv-card-title-vi` | Tiêu đề body font cho tiếng Việt |
| Description | `.gv-card-desc` | Mô tả ngắn trong card |

## Khi nào dùng card variant nào

- Dùng `default` cho phần lớn use-case: docs, list item, settings block, basic content.
- Dùng `dark` khi card nằm trên nền sáng nhưng cần cảm giác nặng, premium hoặc chiều sâu hơn.
- Dùng `gold` cho reward, featured item, highlight state hoặc announcement nổi bật.
- Dùng `hover` khi card có thể click hoặc đóng vai trò như một entry point điều hướng.

## Ghi chú triển khai

- `gv-card-dark` và `gv-card-gold` chỉ đổi theme màu; cấu trúc header/body/footer vẫn giữ nguyên.
- `gv-card-title` phù hợp cho text pixel-style ngắn. Với tiêu đề tiếng Việt dài, nên ưu tiên `.gv-card-title-vi`.
- Nếu card chứa CTA, nên đưa CTA xuống `gv-card-footer` để visual hierarchy rõ hơn.
