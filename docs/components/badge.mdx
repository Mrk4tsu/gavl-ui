# Badge

Badge phù hợp cho status, nhãn phân loại hoặc token ngắn. Base class là `.gv-badge`.

## Khi nào dùng badge

- Dùng khi nội dung rất ngắn: trạng thái, loại item, cấp độ.
- Nếu cần có hành vi hover/click như tag lọc, cân nhắc `.gv-tag` thay vì `.gv-badge`.
