# Vietnamese

GAVL UI được tối ưu riêng cho nội dung tiếng Việt bằng font body và line-height rộng hơn. Đây là điểm quan trọng nếu bạn muốn giữ phong cách pixel nhưng vẫn đọc tốt trên nội dung nhiều dấu.

## Class nên dùng

| Class | Khi dùng |
|---|---|
| `.gv-vi` | Paragraph, content block, mô tả dài |
| `.gv-vi-tight` | Heading tiếng Việt hoặc text ngắn cần gọn |
| `.gv-heading-1` → `.gv-heading-4` | Tiêu đề tiếng Việt |
| `.gv-form-label-vi` | Label form tiếng Việt |
| `.gv-vi-label` | Label ngắn, uppercase mềm |
