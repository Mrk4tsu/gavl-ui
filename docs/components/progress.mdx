# Progress

Progress sử dụng `.gv-progress-wrap` cho track và `.gv-progress-bar` cho phần fill. Width của bar được điều khiển bằng inline style hoặc JavaScript.

## Variants

<div className="docs-demo">
  <div className="docs-progress-block">
    <div className="gv-progress-wrap">
      <div className="gv-progress-bar" style={{ width: '45%' }} />
    </div>
    <div className="gv-progress-wrap">
      <div className="gv-progress-bar gv-progress-success" style={{ width: '62%' }} />
    </div>
    <div className="gv-progress-wrap">
      <div className="gv-progress-bar gv-progress-danger" style={{ width: '28%' }} />
    </div>
    <div className="gv-progress-wrap">
      <div className="gv-progress-bar gv-progress-info gv-progress-striped gv-progress-animated" style={{ width: '74%' }} />
    </div>
  </div>
</div>

```html
<div class="gv-progress-wrap">
  <div class="gv-progress-bar gv-progress-success" style="width: 62%;"></div>
</div>
```
