# Design Tokens

GAVL UI dùng **CSS Custom Properties** (biến CSS) cho toàn bộ design system.
Bạn có thể override bất kỳ token nào bằng cách khai báo lại trong `:root`.

## Typography

GAVL UI dùng 2 font riêng biệt với mục đích khác nhau:

<div className="type-family-grid">
  <div className="type-family-card type-family-card--pixel">
    <span className="type-family-eyebrow">Pixel font</span>
    <h4>SVN Determination Sans</h4>
    <p className="type-family-preview">NHIEM VU MOI</p>
    <code>--gv-font-pixel</code>
    <span className="type-family-note">Dùng cho label, display heading và pixel UI text có hỗ trợ tiếng Việt.</span>
  </div>
  <div className="type-family-card type-family-card--body">
    <span className="type-family-eyebrow">Body font</span>
    <h4>Be Vietnam Pro</h4>
    <p className="type-family-preview">Giao diện rõ ràng, hỗ trợ tiếng Việt đầy đủ.</p>
    <code>--gv-font-body</code>
    <span className="type-family-note">Dùng cho heading tiếng Việt, paragraph và nội dung dài.</span>
  </div>
</div>

:::tip[Quan trọng]
`SVN Determination Sans` hỗ trợ tiếng Việt tốt hơn cho các lớp pixel.
Với paragraph dài hoặc heading cần độ đọc cao, `.gv-heading-*` và body font vẫn là lựa chọn ổn định hơn.
:::

## Override token

Bạn có thể customize toàn bộ theme chỉ bằng cách override biến trong CSS của mình:

```css
/* Đổi màu gold thành xanh dương */
:root {
  --gv-gold:       #4a90d9;
  --gv-gold-dark:  #2d6baa;
  --gv-gold-light: #7ab8f0;
}
```

Hoặc chỉ override cho một component cụ thể:

```css
.my-special-btn {
  --gv-gold: #ff6b35;
}
```
