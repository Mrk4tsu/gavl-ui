# Badge

`Badge` là nhãn nhỏ gọn dùng để truyền đạt ngữ nghĩa ngắn — trạng thái, phân loại, cấp độ, vai trò hoặc metadata. Base class là `.gv-badge`, kết hợp với class variant để chọn màu phù hợp với ngữ nghĩa thông tin.

Badge không có hành vi click — chỉ để hiển thị. Nếu cần interaction (hover, toggle filter), hãy dùng `.gv-tag`.

## Variants

Tất cả 7 variant của badge, mỗi variant mang màu ngữ nghĩa riêng:

<div className="docs-demo">
  <div className="docs-demo-inline">
    <span className="gv-badge gv-badge-primary">Primary</span>
    <span className="gv-badge gv-badge-dark">Dark</span>
    <span className="gv-badge gv-badge-danger">Danger</span>
    <span className="gv-badge gv-badge-success">Success</span>
    <span className="gv-badge gv-badge-info">Info</span>
    <span className="gv-badge gv-badge-purple">Purple</span>
    <span className="gv-badge gv-badge-outline">Outline</span>
  </div>
</div>

| Variant | Class | Màu | Ngữ nghĩa gợi ý |
|---|---|---|---|
| Primary | `.gv-badge-primary` | Vàng | Featured, mới, highlight, quan trọng |
| Dark | `.gv-badge-dark` | Nâu tối | Admin, boss, tier cao, neutral đậm |
| Danger | `.gv-badge-danger` | Đỏ | Khẩn cấp, lỗi, cảnh báo, critical |
| Success | `.gv-badge-success` | Xanh lá | Hoàn thành, active, xác nhận, OK |
| Info | `.gv-badge-info` | Xanh dương | Thông tin, hướng dẫn, neutral |
| Purple | `.gv-badge-purple` | Tím | Premium, VIP, legendary, đặc biệt |
| Outline | `.gv-badge-outline` | Viền nâu | Phụ, ít nhấn mạnh, placeholder role |

```html
<span class="gv-badge gv-badge-primary">Primary</span>
<span class="gv-badge gv-badge-success">Success</span>
<span class="gv-badge gv-badge-danger">Danger</span>
<span class="gv-badge gv-badge-outline">Outline</span>
```

## Pixel Badge

Ghép thêm `.gv-badge-pixel` để đổi font sang kiểu pixel — phù hợp cho tag game item, level label, rarity tier và các yếu tố UI có tính chất retro/arcade.

<div className="docs-demo">
  <div className="docs-demo-inline">
    <span className="gv-badge gv-badge-primary gv-badge-pixel">NEW</span>
    <span className="gv-badge gv-badge-dark gv-badge-pixel">BOSS</span>
    <span className="gv-badge gv-badge-danger gv-badge-pixel">CRIT</span>
    <span className="gv-badge gv-badge-success gv-badge-pixel">WIN</span>
    <span className="gv-badge gv-badge-info gv-badge-pixel">INFO</span>
    <span className="gv-badge gv-badge-purple gv-badge-pixel">LEGEND</span>
    <span className="gv-badge gv-badge-outline gv-badge-pixel">SIDE</span>
  </div>
</div>

```html
<!-- Pixel badge: chỉ thêm gv-badge-pixel vào bất kỳ variant nào -->
<span class="gv-badge gv-badge-primary gv-badge-pixel">NEW</span>
<span class="gv-badge gv-badge-purple gv-badge-pixel">LEGEND</span>
<span class="gv-badge gv-badge-danger gv-badge-pixel">BOSS</span>
```

| Đặc điểm | Badge thường | Pixel Badge |
|---|---|---|
| Font | Body font (Be Vietnam Pro) | Pixel font (SVN Determination) |
| Font size | 11px | 7px |
| Padding | 3px 9px | 4px 8px |
| Use case | UI chung, docs, form | Game UI, retro, arcade |

## Anatomy

| Class | Bắt buộc | Vai trò |
|---|---|---|
| `.gv-badge` | Có | Base badge — luôn phải có |
| `.gv-badge-{variant}` | Có | Chọn màu ngữ nghĩa (primary, dark, danger…) |
| `.gv-badge-pixel` | Không | Bật font pixel cho game-style label |

## Badge vs Tag

| Tiêu chí | Badge `.gv-badge` | Tag `.gv-tag` |
|---|---|---|
| Mục đích | Hiển thị thông tin, nhãn tĩnh | Lọc, toggle, tương tác |
| Click | Không | Có (cursor pointer, hover) |
| Prefix `#` | Không | Có (tự động qua `::before`) |
| Font | Nhỏ hơn (11px) | Lớn hơn (12px) |
| Khi nào dùng | Status, role, metadata | Tag lọc, category filter |

## Khi nào dùng variant nào

- **`primary`** (vàng) — mặc định cho nội dung mới, featured hoặc cần thu hút nhẹ.
- **`success`** (xanh lá) — hoàn thành, kích hoạt, trạng thái tốt, xác nhận.
- **`danger`** (đỏ) — khẩn cấp, lỗi, cảnh báo, cần hành động ngay.
- **`info`** (xanh dương) — thông tin trung tính, đang xử lý, không yêu cầu hành động.
- **`dark`** (nâu tối) — label phụ cần contrast cao, admin, boss, tier cao.
- **`purple`** (tím) — phân loại đặc biệt, VIP, legendary, premium content.
- **`outline`** (viền) — khi muốn badge không quá nổi, phụ trợ hoặc trạng thái pending.
- **`pixel`** — bổ sung vào bất kỳ variant nào khi cần cảm giác retro/arcade.

## Ghi chú triển khai

- Badge là `inline-flex`, phù hợp đặt cạnh text hoặc bên trong flex container.
- Không dùng text quá dài — giới hạn lý tưởng là **1–3 từ** (ví dụ: "Hoàn thành", "VIP", "Level 80").
- Badge có `white-space: nowrap`, không tự xuống dòng.
- Khi hiển thị nhiều badge trong một hàng, bọc bằng flex container với `gap` để kiểm soát khoảng cách.
- Pixel badge có font size 7px — chỉ dùng cho text viết hoa ngắn, không dùng cho tiếng Việt có dấu.
