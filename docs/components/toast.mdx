# Toast

Toast dùng `.gv-toast-container` để neo vị trí và `.gv-toast` cho từng message. Vì đây là CSS library, logic thêm hoặc xóa toast sẽ do JavaScript của ứng dụng xử lý.

## Khi nào dùng toast

- Xác nhận action ngắn: lưu thành công, sao chép, cập nhật xong.
- Không nên dùng cho nội dung dài hoặc action cần người dùng đọc kỹ.
